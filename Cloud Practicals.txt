Practical 6 Creating an application in Dropbox to store data securely

>Download Dropbox>Login
>Create Folder
>Search Dropbox Developer on google>Go to dropbox.com/developers
>Click on App Console in top right corner.
>Create an app
>Permissions>Allow write in all>Connect user login allow all 3
>Settings>Generate Access Token
>Go to Google Colab>colab.research.google.com>Create a new notebook
>Write python SDK code:

!pip install dropbox
import dropbox
dropbox_access_token='' //paste the token in the quotes
dropbox_path="/prac6/aa.txt" //prac6 is the folder name and aa.txt is file name
computer_path="aa.txt"
//create a sample text file aa.txt in desktop and upload the file in colab.

client=dropbox_Dropbox(dropbox_access_token)
print("Successfully connected with Dropbox")
client.files_upload(open(computer_path,"rb").read(), dropbox_path)
print("Files successfully uploaded")

>Go to Dropbox website>Refresh>We will see our folder>Click on the folder and we will see our txt file.
>Go to Appsheet appsheet.com>Login
>Create>App>Start with existing data>Create app>Dropbox source>Select xls file>
>Open demo app. File is uploaded successfully.

Practical 9 Installing and Configuring Dockers in localhost and running multiple images on a Docker
Platform

>Install Docker Desktop, Minikube, Kubectl
>Copy kubectl and put it in minikube folder
>Set the system path>Copy kubectl file path>search Environment Variables in PC control panel>Click Environment Variables>Double-click Admin Path>New>Paste file path>Ok>Double-click System Path>New>Paste file path>Ok
>Open cmd>type kubectl>check if installed
>Open cmd>type minikube>check if installed
>Restart PC
>Open Docker Desktop
>Copy sample container line and paste in cmd>Will pull an image. Image is running.
>Open cmd>type minikube start --vm-driver=docker
>Minikube container image is running in Docker Desktop.
